# 支持的语法

> 以下语法按 **块级语法** 和 **行内语法** 分类。所有语法均可通过转义符 `\` 取消效果（支持转义：`[` `]` `*` `|` `-` `~` `#`）。

---

## 一、块级语法（Block Rules）

块级语法以行为单位识别，相邻的同类型行会合并为一个块。

### 1. 标题

```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

- 支持 `#` ~ `#########`（1~9 个 `#`）
- 标题后的内容自动纳入该标题的缩进区域
- 标题内容支持行内语法（如 `## 二级**标**题`）
- **特殊处理**：若标题下仅有一个列表，输出为 `titledList` 结构而非 `hx`

### 2. 无序列表

```
- 项目一
- 项目二
- 项目三
```

- 行首以 `-` 开头（排除 `---` 分隔线）
- 输出 `<ul><li>...</li></ul>`

### 3. 引用块

```
> 这是引用的内容
> 多行引用
```

- 行首以 `>` 或全角 `＞` 开头
- 输出 `<div class="quote">...</div>`
- 支持嵌套引用

### 4. 分隔线

```
---
----
```

- 一行内只有 `-` 字符且至少 3 个
- 输出 `<div class="sep"></div>`

### 5. 行注释

```
// 这是注释内容
```

- 行首（忽略前导空白）以 `//` 开头
- 输出 HTML 注释 `<!--内容-->`

### 6. 迷你表格

```
|姓名|年龄|
|---|---|
|Au|20|
|Br|38|
```

- 行以 `|` 开头和结尾
- 列以 `|` 分隔
- 包含全 `-` 分隔符的行表示表头与数据的分界，其上方为 `<th>`，下方为 `<td>`
- 输出 `<table><tr>...</tr></table>`

#### 表格单元格颜色

```
|一号线/-c-/blue | 89km |
|二号线/-c-/rgb(255,190,190) | 130km |
```

- 在单元格内容后附加 `/-c-/颜色值`
- 颜色值支持：颜色名（如 `red`）、RGB（如 `255,0,0`）、十六进制（如 `#FF0000`）
- 自动根据背景亮度选择黑/白文字色

### 7. 对齐块

```
.   居中对齐的内容
...   右对齐的内容
```

| 前缀 | 效果 |
|------|------|
| `.   `（点+3空格） | 居中对齐 `<div style="text-align:center">` |
| `...   `（3点+3空格） | 右对齐 `<div style="text-align:right">` |

### 8. 脚注定义

```
[^脚注名]: 脚注内容
```

- 行首匹配 `[^名称]:` 或 `[^名称]：`
- 脚注内容支持行内语法
- 脚注本体在文档末尾统一输出

## 二、行内语法（Inline Rules）

行内语法在段落内部识别，可嵌套使用。

### 1. 文本样式

| 语法 | 效果 | HTML |
|------|------|------|
| `**粗体**` | **粗体** | `<b>` |
| `*斜体*` | *斜体* | `<i>` |
| `~~删除线~~` | ~~删除线~~ | `<s>` |
| `***下划线***` | 下划线 | `<u>`（不推荐，用 `~~`） |
| `****删除线****` | 删除线 | `<s>`（不推荐，用 `~~`） |

### 2. 上下角标

| 语法 | 效果 | HTML |
|------|------|------|
| `_(下标)` | 下标 | `<sub>` |
| `^(上标)` | 上标 | `<sup>` |
| `\sub内容\sub` | 下标 | `<sub>`（不推荐） |
| `\sup内容\sup` | 上标 | `<sup>`（不推荐） |

### 3. 行内代码

```
这是 `code` 代码
```

- 语法：`` `代码` ``
- 输出 `<code>代码</code>`

### 4. 链接

#### 4.1 自动链接

```
[词条名]
[/path/to/page]
[https://example.com]
```

- 若 `词条名` 在系统中存在，通过 `IScopedConvertingProvider.Link` 解析为链接
- 若不存在，输出红链 `<a class="redLink" pathName="词条名">词条名</a>`
- URL 直接输出 `<a href="...">...</a>`

#### 4.2 带文字的链接

```
[显示文字](目标)
[武汉](wuhan)
[百度](https://baidu.com)
```

- 目标在系统中存在时解析为链接
- 不存在时输出红链

### 5. 行内媒体对象

```
[http://img.png]
[http://img.png|8]
[http://img.png|8|left]
[http://img.png|8|left|>800]
[http://img.png|8||<800]
[http://img.png|8|block|800-1200]
[http://audio.mp3]
[http://video.webm]
```

- 格式：`[URL|高度|命令|媒体查询]`
- 高度：纯数字自动加 `em`，也可指定 `px`
- 命令：`left`（左浮动）、`block`（块级居中）、`inline`（行内），默认右浮动
- 媒体查询（可选）：默认显示，仅在指定视口宽度下保留；不满足条件时隐藏
  - `>800` 或 `＞800`：宽度大于 800px 时显示，小于等于 800px 时隐藏
  - `<800` 或 `＜800`：宽度小于 800px 时显示，大于等于 800px 时隐藏
  - `800-1200`：宽度在 800px 到 1200px 之间显示，否则隐藏
- 根据文件扩展名自动识别：图片（`img`）、音频（`audio`）、视频（`video`）；`svg` 图片使用 `<object>` 渲染

### 6. 彩色文本 / 彩色块

```
#red\@红色文字#
#255,0,0\@红色文字#
#FF0000\@红色文字#
#red#        ← 纯颜色块（无文字）
```

- 含 `\@`：彩色文字，左侧为颜色值，右侧为文字内容
- 不含 `\@`：彩色方块（宽高由字体大小决定）
- 颜色值支持：颜色名、RGB、十六进制
- 若颜色值无效，原样保留 `#...#`

### 7. 特殊效果

| 语法 | 效果 | 说明 |
|------|------|------|
| `\bd 内容 \bd` | 带边框文字 | `<span class="bordered">` |
| `\hd 内容 \hd` | 黑底 hover 显示 | 鼠标悬停显示文字 |
| `\ct 内容 \ct` | 居中文本块 | `<div style="text-align:center">` |
| `\mq 内容 \mq` | 滚动字幕 | `<marquee>` |
| `\滚 内容 \滚` | 滚动字幕 | `<marquee>`（中文别名） |

### 8. 脚注引用

```
这是一个脚注引用[^1]
```

- 在行内使用 `[^脚注名]`
- 与块级脚注定义 `[^脚注名]: 内容` 配对
- 输出上标链接 `<sup><a id="refentry_名称">[名称]</a></sup>`

### 9. 植入 / 短链接

```
{词条名}
```

- 通过 `IScopedConvertingProvider.Implant` 解析
- 若解析失败，保留原样 `{词条名}`
- 常用于在 style 标签等场景中保留 `{}`

### 10. 模板调用

```
{{模板名}参数值}
{{模板名} 参数名::参数值 && 参数名2::参数值2}
```

- 外层 `{}` 包裹整个调用，内层 `{模板名}` 指定模板
- 参数分隔：`&&`
- 键值分隔：`::` 或 `：：`
- 若只有一个无名参数，自动匹配第一个插槽

## 三、自动替换

自动替换是在解析前将特定文本替换为 HTML 的功能，通过 `IScopedConvertingProvider.Replace` 实现。

```
更多有趣内容见3C教育体系大纲等词条
→ <p>更多有趣内容见<a href="/w/6">3C教育体系大纲</a>等词条</p>
```

- 替换目标在 `ParserBuilder.AutoReplace` 中配置
- 支持 `SingleUse`（单次替换）和持续替换两种模式
- 替换目标作为特殊的 `LiteralInlineRule` 参与行内解析

---

## 四、转义规则

以下字符前加 `\` 可取消其特殊含义：

```
\[  \]  \*  \|  \-  \~  \#
```

示例：

```
\*这不是斜体\*
→ *这不是斜体*
```

---

## 五、HTML 安全

- 所有文本在分行时经过 `HtmlSanitizer` 编码
- 模板源码中的 HTML 标签需要自行确保安全性
- 带 JavaScript 的模板仅允许超级管理员创建（防止 XSS）
