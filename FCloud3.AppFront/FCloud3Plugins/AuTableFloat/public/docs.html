AuTableFloat 表格浮动插件

在表格的任意单元格中写入触发词，即可让该表格浮动到右侧，并自动移除包含触发词的那一行。

触发词

  • au-table-float

基本用法

在表格任意单元格中填入触发词：

au-table-float
第一列数据第二列数据
注意：包含触发词的行会在页面渲染后被自动移除，用户看不到这一行。

效果

  • 触发词所在的整行会被移除。
  • 触发词所在的 <table> 元素会被设置为 float: right，并添加类名 au-floated-table。
  • 表格会被移动到当前页面第一个 .wikiView 元素内的第一个 .paras 元素内部的开头位置。
  • 被移动到其前面的 .paras 元素会添加类名 au-floated-paras。
  • 插件运行时会自动向 <head> 末尾插入 <style id="au-table-float-styles">，为上述类名提供默认样式；多次运行不会重复插入。

注入样式

<style id="au-table-float-styles">
    .au-floated-table {
        margin: 10px !important;
    }
    .au-floated-paras .para,
    .au-floated-paras .indent {
        overflow-x: unset;
    }
</style>

错误处理

如果页面中不存在 .wikiView .paras 元素，表格仍会被移除触发行并设置浮动样式，同时会在表格后方显示红色错误文本：找不到.paras元素。

注意事项

  • 触发词需要独占单元格内容（去除前后空白后完全匹配）。
  • 如果一行中有多个单元格包含触发词，该行也只会被移除一次。
  • 如果同一个表格中有多个行包含触发词，这些行都会被移除。
  • 插件按表格独立处理，多个表格互不干扰。
  • 只使用页面中第一个 .wikiView 元素内的第一个 .paras 元素。
  • 触发词区分大小写，必须使用 au-table-float。
