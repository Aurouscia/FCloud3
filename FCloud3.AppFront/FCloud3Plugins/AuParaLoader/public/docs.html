AuParaLoader 段落加载插件

从其他词条加载指定段落内容，嵌入到当前页面中。

主要用途：多个词条共用一个段落（确保内容完全一致）可站内或跨站

建议：专门建立词条放置这种共享段落

触发词

以下任意一个均可触发：

  • AuParaLoader
  • au-pl
  • au-para-loader

基本用法

在单行单列表格中写入触发词，括号内填入参数：

AuParaLoader(目标词条路径)
注意：表格会被替换为加载的内容，原表格不再可见。

参数格式

参数用逗号分隔，最多三个：

目标词条路径必填，要加载的词条路径名
段落选择器可选，按标题包含文字匹配段落
URL 基础路径可选，自定义 API 基础路径

完整示例

加载词条第一个段落

AuParaLoader(help)

加载指定段落

AuParaLoader(help, 快速开始)

使用自定义域名（跨站）

AuParaLoader(help, 快速开始, https://example.com)

注意事项

  • 表格必须是单行单列，否则不会被识别。
  • 目标段落通过标题包含文字匹配，不需要完整标题。
  • 未指定段落选择器时，加载词条的第一个段落。
  • 加载失败时表格位置会显示红色错误信息。
  • 跨站加载时，指定域名必须也部署着 fcloud3 网站。
  • 如果跨站加载失败，请联系网站管理员。
  • 触发词区分大小写。
