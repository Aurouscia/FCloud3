AuRouteRenderer 线路图渲染插件

将表格中的字符指令渲染为 Canvas 地铁/铁路线路图。

触发词

以下任意一个均可触发：

  • AuRtRd
  • au-rr
  • au-route-renderer

实际案例

由于本插件较为复杂，建议查看有实际效果对比的页面
可参考：http://wiki.jowei19.com/#/w/in-wiki-route-renderer

基本结构

在表格第一列的单元格中写入指令，第二列写站点名称。首行必须包含触发词，可附带颜色配置。触发词前后可以有其他普通表格行，它们不会被渲染：

AuRtRd(c:#009999)1号线
o ; 5(#ffff00)起点站
o\T;K(#f00)换乘站

配置参数

触发词后可跟 (键:值_键:值) 格式的配置，下划线分隔多个键值对。括号必须为英文半角括号：

键说明示例
c线路颜色（CSS 颜色值）c:#009999

绘制指令字符

每个单元格的第一部分（分号前）由以下字符组成网格，描述线路走向：

字符名称说明
o站点绘制一个车站圆点
T线路绘制连接上下站点的轨道线段
_空位占位，不绘制任何内容
/左换乘线路向左偏移的换乘/转弯连接
\右换乘线路向右偏移的换乘/转弯连接
~水域绘制河流/水域横线
^山丘绘制山丘/地形横线
b上分支从当前位置向上方绘制分支箭头
p下分支从当前位置向下方绘制分支箭头
B双向分支同时绘制上下两个分支箭头
n上转弯线路向上转弯（配合 - 使用）
u下转弯线路向下转弯（配合 - 使用）
-转弯跨度连接 n 或 u 的横向跨度填充

标注（Annotation）

分号后的部分为标注，可附加在站点旁显示图标或文字：

格式说明示例
文字(颜色)带背景色的圆形图标，最多显示3个字K(#f00) 显示红色背景 "K"
_文字_灰色背景的固定图标_机场_ 显示飞机图标
_airport_特殊处理：绘制飞机图案_airport_

多个标注用分号 ; 分隔。奇数位置的标注显示在站点上方，偶数位置显示在下方。

完整示例

基础线路

触发词和第一个站点可以写在同一个单元格内，用换行隔开：

AuRtRd(c:#009999)
o ; 5(#ffff00)		1号线
o中间站
o终点站

也可以分开放置：

AuRtRd(c:#009999)1号线
o ; 5(#ffff00)起点站
o中间站
o终点站

换乘站

AuRtRd(c:#77aaff)3号线
o\T;浑(#009900);Au(#000044);L23(cornflowerblue)多线换乘站
o/下一站

含地形与分支

AuRtRd(c:#aa6666)2号线
oo;_airport_天河机场
~~~长江
oTb;积玉桥（上分支）
T^^洪山（山丘）
u-u街道口（下转弯）

注意事项

  • 触发词必须出现在表格第一列的首个有效单元格中。
  • 每个绘制单元格的第一部分只能包含上表列出的有效字符。
  • 站点 o 和线路 T 可以组合使用，如 oTo 表示站点-线路-站点。
  • 换乘标记 /、\ 会自动根据相邻单元格计算连接方向。
  • 表格会被替换为 Canvas 渲染的线路图，用于标注线路的单元格不再可见。
